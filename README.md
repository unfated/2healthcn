# 2healthcn

Official corporate website for **2Health AI** (`2HEALTH AI LIMITED`, Hong Kong), served at **[2health.cn](https://2health.cn)**.

This is the company's public marketing site — distinct from the product platform at **[2health.ai](https://2health.ai)**.

## Contents

- `index.html` — the complete, self-contained website (single file, no build step, no external dependencies).

## Design

- **Sections:** Hero → About → Platform → People → Contact.
- **Identity:** petrol-teal accent on a teal-tinted neutral, a serif / sans / mono type system, and a live biosignal canvas in the hero.
- **Themes:** respects the visitor's OS light/dark preference and includes an in-page theme toggle.
- **Accessibility:** semantic landmarks, visible focus states, and `prefers-reduced-motion` support.
- **Self-contained:** all CSS, JavaScript, and graphics are inline — open `index.html` in any browser, or host it on any static host.
